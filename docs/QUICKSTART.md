# Quickstart — escolha o seu

O ROLDAO-METHOD tem dois pontos de entrada — pra você não tropeçar em jargão que não é pra você:

## Não programa?

Você é dono do produto, idealizador, gestor — e quer entender o que esse framework te dá sem precisar saber o que é "hook" ou "commit".

→ **[`docs/QUICKSTART-DONO.md`](QUICKSTART-DONO.md)** (5 minutos, sem jargão).

## É desenvolvedor?

Você vai mexer no projeto, configurar IDE, integrar addons, conferir hooks.

→ **[`docs/QUICKSTART-DEV.md`](QUICKSTART-DEV.md)** (5 minutos, técnico completo).
