# v0.20.0

## O que mudou

Esta release fecha 5 itens adiados da rodada anterior e arruma o CI que vinha vermelho há semanas. O framework ganha um 15º agente (Lucas, DevOps/infra) e uma 13ª skill BR (validador de código IBGE de município). O instalador ficou mais curto, o pitch do README mais escaneável, e todos os 19 jobs do CI passam.

## Por que importa

- **Cobertura de infra** — Lucas cuida de CI/CD, deploy, observabilidade e secrets em cloud BR (sa-east-1, southamerica-east1, brazilsouth). Recusa apply destrutivo sem confirmação humana (SEC-002 + INV-AGENT-005). Quem integra Pix, NF-e, eSocial agora tem um especialista pra revisar pipeline e rollout antes de ir pra prod.
- **NF-e + eSocial mais confiáveis** — `validar-codigo-municipio-ibge` evita reprovação por código de município com DV inválido (SEFAZ recusa silenciosamente). Antes só `validar-chave-acesso-nfe` cobria validação fiscal por DV.
- **Onboarding mais limpo** — quem instala vê 3 próximos passos diretos, não 8. Pitch do README mostra o diferencial na primeira tela.
- **CI verde dá confiança pra próximos releases** — 9+ push consecutivos vinham vermelhos por débito acumulado (encoding flaky, hook bit step quebrado, etc.). Tudo resolvido em 7 commits cirúrgicos.

## Como aplicar

1. `npx roldao-method update` na raiz do projeto.
2. Confirme o que mudou: o instalador faz backup `.bak` de cada arquivo do framework antes de sobrescrever. AGENTS.md, CLAUDE.md, REGRAS-INEGOCIAVEIS.md e settings.local.json são preservados sem tocar.
3. Se você ainda não viu, copie `templates/CLAUDE.local.md.example` → `CLAUDE.local.md` na raiz pra preferências pessoais (gitignored). MCP preset BR fica em `.mcp.json.examples/`; GitHub Action de code review em `templates/.github/workflows/claude-review.yml`. Detalhes em `docs/QUICKSTART.md §8 Opcionais`.

## Atenção

- Nenhum breaking change. Comportamento dos 26 hooks bloqueadores intacto (179/179 testes verdes).
- Se você tinha customizações no `tools/sincronizar-claude-md.js`, revise — agora o tool aceita ausência da raiz (era bug que travava CI).
